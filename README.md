mruby-lingrbot
==============

mruby bot on lingr

install
-------

    $ git clone https://github.com/mruby/mruby
    $ git clone https://github.com/mattn/mruby-lingrbot
    $ cp mruby-lingrbot/*.rb mruby/.
    $ cd mruby && rake
    $ bin/mruby mruby-lingrbot.rb
